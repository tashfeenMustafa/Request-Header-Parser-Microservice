Request Header Parser Microservice API

This is a Microservice API that returns your browser's IP address, the language it uses and its operating system.

Usage:

https://request-header-parser-microservice-tashfeenmustafachoudhury.c9users.io/whoami

Example Response:

{"ip":"103.60.175.118","language":"en-US","software":"X11; Linux x86_64"}
